# Literals

> **char**
    eg: `&#96;a&#96;`, `&#96;\0&#96;`

> **int**
    eg: `6`, `0x7f`, `0b1001`, `0o7332`

> **sint**
    eg: `-43`, `+2`

> **float**
    eg: `20.3`, `-0.002`

> **str**
    eg: `"hello, world!"`, `hi\b\0`

> **array/struc**
    Exact type is decided contextually.
    eg: `(1,3,9,0,1,3)`, `(2, "hi mom!", 2.0)`
